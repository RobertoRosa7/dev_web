Estrutura de Repetição parte I

Usando estrutura de repetição para percorrer a estrutura árvore (DOM) do HTML para verificar as ocorrência e analisar todo o documento.
Com a estrutura de repetição, podemos criar um bloco de código e modificar somente seu valor e assim ter uma lista do mesmo código, evitando assim, ter que escrever todo o código.
Para toda repetição é necessário ter um fim, se não entraria em loop infinito, portanto, para todo laço de repetição é obrigatório ter um ponto final, uma condição que diga a intrução que quando chegar numa determinada condição é para parar.
