Variáveis JavaScript

Varáveis Introdução

O que são variáveis?

Tipos de variáveis = String - Number - Boolean

String = (str) conjutnto de caracteres

Number = (int ou float) reservador para números inteiro ou reais

Boolean = false ou true - verdadeiro ou falso

Regras para declração de variáveis

Não podem ser iniciados com números, apenas com letras ou "_"

Não podem conter caracteres especiais com: "ç, &, @ etc..."

Não podem ser iguais as palavras reservadas pela linguagem

Declaração de variáveis com "var"

Exemplos:

var nome, var AutoVariavel, var _idade

Exemplo Errado:

var 123nome, var número

Variáveis null e undefined

São valores nativamente do javascript, assim como boolean, number e string. A diferença representa a intenção de algum valor, isto é, uma variável existente mas sem valor no seu endereço de momória.

Undefined, realmente não possui nenhum valor na variável

O que é concatenação

Concatenação é juntar os valores que estão nas variáveis, de forma a montar string ou textos personalizados. Permite aplicar determinados contéudo decorrente da aplicação.
