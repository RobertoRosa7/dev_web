Introdução ao if

if(condicional){code}

A instrução if é um condicional com instruções de código para serem executadas

Operadores de comparação

Também conhecido como operadores relacionais, servem para formar as expressões condicionais do comando if

Operadores

(==) igual verifica se os valores comparados são iguais

(===) idêntico verifica se os valores comparados são idênticos

(!=) diferente verifica se os valores comparados são diferentes

(!==) Não idêntico verifica se os valores comparados são diferentes e de tipos diferentes

(<) menor que verifica se os valores comparados é menor que

(>) maior que verifica se os valores comparados maior que

(<=) igual ou menor verifica se os valores comparados igual ou menor que

(>=) igual ou maior verifica se os valores comparados igual ou maior que
