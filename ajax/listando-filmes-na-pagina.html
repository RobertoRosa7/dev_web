Listando os filmes na página

Passo 01:

Para listar os filmes é preciso usar um laço de repetição, mas antes deve ser feito uma estrutura HTML para receber os valores convertido para JSON

Passo 02:

A primeira coisa que deve ser fazer é percorrer o arquivo convertido para JSON, e acessar os índice do array, neste caso, será um array multidimensional.

Passo 03:

Depois de percorrer os índice do array é hora de associar cada elemento com as tags HTML, para isso é preciso criar um novo elemento de forma programatica pelo JavaScript que conterá os conteúdos. Este novo elemente deve estar dentro do laço de repetição.

Passo 04:

Após o termino de criação dos elemento de forma programática é hora de associar os conteúdo do JSON recente convertido

Passo 05:

Adicionar o conteúdo na div container como sendo esta - a div pai. Usando appendChild
