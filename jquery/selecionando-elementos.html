Selecionando elementos parte 2

Selecionado elementos por classe de tag específica

Para fazer uma selecionar um elemento pela classe, isto é, todos os elementos que possuem a mesma classe deve suar $('.class')

Agora selecionar um elemento específico que esteja usando a mesma classe que foi selecionada deve cololar o nome da tag antes do ponto da classe - $('div.class') | $('li.class') | $('input.class')

Selecionando elemento por classe de classe específica

Adicionar uma outra classe na div superior para poder diferenciá-la e, assim poder selecionar todos os elementos que são filhos da classe diferenciadora e que estão usando a mesma classe - $('.lista .exemplo2')

Atuação sobre array de retorno

A primeira coisa que deve ter em mente que sempre quando usamos a seleção do JQuery o retorno da seleção é um Array - atráves da intrução :even |:odd | :first | :last - podemos fazer seleção dentro do array

  • (:even) selecionar dentro do array os índices pares - $('.lista2 .exemplo2:even)
  • (:odd) selecioanr dentro do array os índices impares - $('.lista2 .exemplo2:odd)
  • (:first) selecionar o primeiro elemento dentro do array - $('.lista2 .exemplo2:first)
  • (:last) selecionar o último elemento dentro do array - $('.lista2 .exemplo2:last)
