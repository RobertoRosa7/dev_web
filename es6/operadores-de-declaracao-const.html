Operador Const

Const

O operador const, é uma constante de variável que não altera seu valor no decorrer do script. Para as declaração é uma boa prática usar caracteres maiúsculos.
Diferente de declarações comuns de variáveis as const deve receber seu valor no momento de sua declaração.
O escopo global não interfere no escopo da função, portanto, a const pode ser aplicadas nas funções e vise versa.

Normalmente, as constantes são usada nas declarações de senhas, logins, ou qualquer outro valor que são constantes no decorrer do código, assim, toda aplicação terá as mesmas constante independemente se houver repetições.
