Template String

É uma interpolação da interpretação do javascript. Para fazer uma simplificação de concatenação das string quando é exibida ao usuário, assim com o uso do template string é mais eficaz e traz melhor resultado na hora de codificar. Para o uso do template é preciso usar cráse(``), não é aspas simples.

Para a inserção de variáveis dentro do template string é necessário o uso de ( ${}, cifrão mais chaves)
