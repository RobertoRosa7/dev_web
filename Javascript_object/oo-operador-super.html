Operador super()

Quando um objeto é instanciado, e não recebe nenhum parâmetro, será configurado os valores que estiverem no constructor, assim, o operador super(), fornece acesso a classe pai, com isso ao passar parâmetros no super() será recebido na classe pai. Uma classe filho pode passar parâmetros para a classe pai, modificando assim toda uma cadeia de classes que são extendida pela classe pai. O super() dá acesso ao constructor da classe pai.
