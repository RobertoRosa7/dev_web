Objetos Literais

Objetos literias são criados porque nem sempre será necessário criar um classe para poder usá-lo, assim podemos criar objetos de forma literal.
Para o uso do dia a dia, é conveniente usar objetos literais devido a sua simplicidade e códigos pequenos Normlamente são único dentro do projeto, não necessidade de criar um coleção desse tipo de objeto, até porque se é necessário deve usar classes.
