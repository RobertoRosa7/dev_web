Convertendo XML para JSON

O XML não é suportado pelo JavaScript, portanto, deve ser convertido para JSON que é nativamente suportado.
Como XML é um arquivo de texto para padronização de documento e portanto, é público, devemos convertê-lo para poder realizar as tratativas que será aplicado no aplicação. Convertê-lo para um objeto, e em seguida percorrer todos os índice realizando as devidas configurações.

O primeiro passo é baixar um script JavaScript que será responsável pelo parse, isto é, a conversão do XML, no DOM existe um classe - chamado "DOMParse" devemos instanciá-lo para fazer uso de seus métodos.
