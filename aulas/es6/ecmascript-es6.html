O que é ECMAScript?

Netscape

1995 - Criação por Brendam Eich para o navegador 2

1997 - ECMAS (European Computer Manufactore Association)

Microsoft

1996 - JScript, criação para IE 3

ES1 - ES2 - ES3 - ES4 - ES5 - ES6 - ES7

O ECMAS é uma forma de padronização do JavaScript para diferentes navegadores, assim de uma forma geral todos os browsers poderam exibir e interagir da mesma forma independemente de qual browser ou sistema operacional está rodando.
