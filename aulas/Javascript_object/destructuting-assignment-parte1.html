Destructing Assignment

Desistruturação, retirada de valores dentro de uma estrutura, neste caso foi feito através de um Array assim, para fazer uma destruturação é preciso indicar declaração de variáveis, (let ou const) seguido do colchetes.

Caso precise pular algum índice, basta usar espaço dentro do destructing e se caso adicionar novas variávies a mais, será definido como (undifined). Dá também de adicionar valores padrões dentro do destructing
