Funções Construtoras - Encapsulamento

O príncipio de encapsulamento consiste na aplicação de camadas de proteção aos atributos de classes, como sendo um dos pilares da programação orientada a objetos, no javascript podemos fazer algo parecido usando unções construtoras devido a uma convenção estabelecida
Para fazer isso, basta utilizar operador de declaração variáveis (var) dentro do escopo da função, o que normalmente é usado (this).
